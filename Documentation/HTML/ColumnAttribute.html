The ColumnAttribute class is, not unsurprisingly an implementation of the Attribute class, which can be applied to class members of an assembly. In practice it is only applied to classes inheriting from the XPTable.Column class although that rule is not specifically implemented. The attribute serves three purposes:-

  1. Its presence indicates on a class definition will cause that class to appear in the drop down list of types in the ColumnCollectionEditor. The name displayed in that list is derived from the Type name of the Column class.

  1. It contains information pertinent to DataSource columns, specifically the property name from which a cells value is obtained.

  2. It contains constraints and default values pertinent to the ColumnWidth property of the Column.
