This class provides the ability to control the sequence in which the properties within an expandable object are displayed. It is often the case that the order in which properties are displayed in the PropertyGrid Category Tab are ordered inconveniently unless they are given contrived names or numerical prefix's. The order is specified by the PropertyGrid.PropertySort property, which is an enumerated type having 4 possible values

Alphabetical

Categorized

CategorizedAlphabetical

NoSort

The PropertyGrid toolbar provides a PropertyTab for two of these values, Alphabetical and CategorizedAlphabetical. PropertySort.Categorized (and the author suspects the same is true of PropertySort.NoSort) provides the means via which a TypeConverter's GetProperties method can determine the sort order. The author is yet to determine why PropertyGrid explicitly does not provide these options in its toolbar, surely not the absence of a PropertyOrderAttribute.

Obtains the PropertyDescriptorCollection from TypeDescriptor.GetProperties for the _value and _attributes specified. The PropertyDescriptors are interrogated to find any PropertyOrderAttribute which is used to form a sortable collection of name, integer pairs from which a string array of Property Names can be extracted for use by the PropertyDescriptorCollection.Sort(string[] names) method.
