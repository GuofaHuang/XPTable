This class provides an augmentation to XPTable, in the form of ToolStrip and an optional companion ContextMenuStrip.



public abstract class TableEventCancel<TTData> where TTData : System.ComponentModel.CancelEventArgs

[ToolboxItem(true)]

[ToolboxBitmap(typeof(ToolStrip))]

public sealed class TableActionStrip : ToolStrip, ISupportInitialize

The ToolStrip is a collection of buttons that correspond to values enumerated by TablesActions, an option is provided to create a corresponding ContextMenuStrip to be assigned to the Tables ContextMenuStrip property. The TableActionStrip is designed to be hosted within one of the peripheral panels of a ToolStripContainer (ie the Left, Right, Top or Bottom Panels) with the associated Table docked in the ToolStripContainer's Content panel. In design mode the button images will always be those designed for the Left ToolStrip Panel irrespective of which panel the TableActionStrip occupies. however at run time the images are adjusted to reflect the actual panel in which the TableActionStrip is contained.
