This class implements an attribute that can be attached to Properties in oder to specify the order of the properties in an expandable property which specifies the TypeConverter SortedExpandableObjectConverter.

[AttributeUsage(AttributeTargets.Property, AllowMultiple=false, Inherited=true)]

public class PropertyOrderAttribute : Attribute

SortOrder

{read only)

Int

An integer value representing the order in which the property is to be shown. It is suggested that Properties be given SortOrder values that increment by something larger than 1 (e.g. 5) so that new properties can be more easily introduced.
