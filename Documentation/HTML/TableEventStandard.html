public abstract class TableEventStandard<TTData> where TTData : System.EventArgs

Provides a abstract generic class for standard events (i.e. those that cannot be cancelled). It encapsulates the essential components required to process events, the event handler definition, the core event and the OnEvent method, in addition it provides a lock over the event when delegates are added to or removed from the event or the delegates are being invoked when the relevant event is signalled via the OnEvent method (in actual fact the lock is only held whilst the handler list is copied, it is this copy that is used to invoke the handlers, this is sone to prevent wedging if a handler happens to spin into a long running process).

Usage


A class needing an event would typically delare a nested class as follows


public MyClass

{

public class CounterChangedEvent :TableEventStandard<MyEvent.Data>

{

public class Data : System.EventArgs

{

public int OldCounter { get {return this.theOldCounter; } }


internal Data(int _oldCounter)

{

this.theOldCounter = _oldCounter;

}

private int theOLdCounter;

}

internal void Raise(object _sender, int _oldCounter)

{

this.OnEvent(_sender, new Data(_oldCounter));

}

}


Note the Raise method is not enforced, it is a convention adopted by the developer. Inheritors can use another name or they may choose to override the OnEvent member. The rationale of using a “local” method rather an OnEvent override is that the latter would require the “event raising code” to constructs Data object. The author finds it clearer to have “event raising code” invoke a local method with direct arguments rather than via a the data block constructor.

An instance of the class can be declared and instanced thus


private CounterChangedEvent theCounterChanged;


External access to the CounterChanged event can be provided thus


public event CounterChangedEvent.Handler CounterChanged

{

add{ this.theCounterChangedEvent.Event += value; }

remove{ this.theCounterChangedEvent.Event -= value; }

}


The event is triggered as in :


public int Counter

{

get { return this.theCounter; }

set

{

if (value != this.theCounter)

{

int __oldCounter = this.theCounter

this.theCounter=value;

this.theCounterChangedEvent.Raise(this, __oldCounter);

}

}
